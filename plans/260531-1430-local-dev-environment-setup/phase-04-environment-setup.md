## Requirements

| Variable | Value |
|----------|-------|

| DB_DATABASE | vnct |

| DB_PASSWORD | (empty or root) |

### Other Required Variables

| Variable | Purpose |
|----------|---------|
| APP_NAME | VNCT |
| APP_ENV | local |
| APP_KEY | Generated by Laravel |
| APP_DEBUG | true |
| APP_URL | http://localhost:8000 |

## Implementation Steps

### Step 1: Create .env file manually

**IMPORTANT:** This project does NOT have `.env.example`. You must create `.env` manually.

```bash
cd /Users/hieuspace/Desktop/CODE/vnct-vietnamconnect/vnct
```

Create `.env` file with the template below (Step 3).

### Step 2: Generate Laravel Application Key

```bash
php artisan key:generate
```

This sets `APP_KEY=` in `.env`.

### Step 3: Configure Database Connection

Edit `.env`:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=vnct
DB_USERNAME=root
DB_PASSWORD=
```

### Step 4: Verify Configuration

```bash
php artisan config:clear
php artisan config:show database
```

## .env Template (if .env.example missing)

```env
APP_NAME=VNCT
APP_ENV=local
APP_KEY=
APP_DEBUG=true
APP_URL=http://localhost:8000

LOG_CHANNEL=stack
LOG_LEVEL=debug

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=vnct
DB_USERNAME=root
DB_PASSWORD=

BROADCAST_DRIVER=log
CACHE_DRIVER=file
FILESYSTEM_DRIVER=local
QUEUE_CONNECTION=sync
SESSION_DRIVER=file
SESSION_LIFETIME=120

MAIL_MAILER=smtp
MAIL_HOST=mailhog
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS=hello@example.com
MAIL_FROM_NAME="${APP_NAME}"
```

## Failure Modes

| Failure | Diagnosis | Fix |
|---------|-----------|-----|
| APP_KEY empty | key:generate failed | Manually set key with `php artisan key:generate --force` |
| DB connection fails | MySQL not running | `brew services start mysql@8.0` |
